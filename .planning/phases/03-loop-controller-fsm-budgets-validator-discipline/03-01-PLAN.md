       - Invokes `loongforge-phase-gate --run-dir {run_dir} --phase {phase}` via subprocess.run with 300-second timeout.
       - If "PASSED" in stdout: status="passed", failure_signature=None.
       - If "BLOCKED" in stderr: status="failed". Attempt to parse structured failure from phaseN_output.yml validator block: if validator dict has "failure_signature" with "kind" and "location" keys, construct FailureSignature. Otherwise failure_signature=None (VAL-02 free-text-only).

    5. `should_rerun_for_flake(result: ValidatorResult, phase: int) -> bool`:
       Return True iff: phase in FLAKE_RERUN_PHASES AND result.status == "failed" AND result.failure_signature is not None AND result.failure_signature.kind in ("numerical_mismatch", "threshold_exceeded").

    6. `check_validator_integrity(run_dir: Path, phase: int, attempt_start_time: str, recorded_hash: str|None = None) -> dict`:

       (1) If recorded_hash: compute sha256(run_dir / "bin" / "loongforge-phase-gate".read_bytes())[:16]. Compare to recorded_hash.
       (2) log_dir = run_dir / "phases" / f"phase{phase}" / "logs". Check exists and any files.
       (3) If logs present, parse attempt_start_time as ISO datetime. Check at least one log file has st_mtime >= attempt_start_time.timestamp().
       integrity_ok = all three checks pass.

    7. `get_megatron_head_sha(gh: GhClient, owner_repo: str, ref: str) -> str`:
       Call `gh._run(["api", f"repos/{owner_repo}/git/ref/heads/{ref}", "--jq", ".object.sha"])`. Return stdout.strip() on success (returncode==0). Raise RuntimeError on failure.
