    **check_budget** function:
    ```python
    def check_budget(
        budget: LoopBudget,
        phase_attempts: int,
        total_attempts: int,
        run_start_time: str,
    ) -> ExitReason | None:
        """Return ExitReason if any budget axis is breached, else None.
        IMPORTANT: This check MUST happen before processing validator results.
        If budget is breached, exit reason is ALWAYS EXHAUSTED, never passed (Pitfall 2)."""
        if phase_attempts >= budget.max_attempts_per_phase:
            return ExitReason.EXHAUSTED
        if total_attempts >= budget.max_attempts_per_run:
            return ExitReason.EXHAUSTED
        from datetime import datetime, timezone
        start = datetime.fromisoformat(run_start_time)
        elapsed = (datetime.now(timezone.utc) - start).total_seconds() / 60.0
        if elapsed >= budget.max_wallclock_minutes:
            return ExitReason.EXHAUSTED
        return None
    ```

    **Main function:**

    - `run_phase_loop(run_dir: Path, phase: int, gh: GhClient, budget: LoopBudget, dry_run: bool = False, max_iterations: int = 100) -> ExitReason`:
      Re-entrant controller implementing LOOP-01 FSM dispatch:

      1. Read state: `state = LoopState.from_disk(run_dir, phase)`

      2. Record validator_hash at first entry if not recorded: if state.validator_hash is None and (run_dir / "bin" / "loongforge-phase-gate").exists(), compute sha256 and store via dataclasses.replace.

      3. Budget pre-check (LOOP-03, Pitfall 2): `budget_breach = check_budget(budget, state.attempt, state.total_attempts_used, state.run_start_time)`. If breached: _transition to EXIT, append budget_check row, persist, return EXHAUSTED.

      4. Safety iteration limit: if max_iterations <= 0, return EXHAUSTED.

      5. FSM dispatch via match/case on state.current_state:
         - `FSMState.PROBE`: _transition to EDIT with kind="probe". Persist. Recurse with max_iterations-1.
         - `FSMState.EDIT`: _transition to PR with kind="edit". Persist. Recurse.
         - `FSMState.PR`:

If None, skip PR/issue steps and transition to VALIDATE directly (local-only mode).
           - If repos_info provided: branch_name = f"adapt/{run_id}/phase{phase}/attempt{state.attempt}", owner_repo = repos_info["loongforge_repo"], base_ref = repos_info["loongforge_base_ref"].
           - Call `gh.open_pr(owner_repo, head=branch_name, base=base_ref, run_id=..., phase=phase, attempt=state.attempt, kind="base")`.

           - _transition to VALIDATE with kind="merge_base". Persist. Recurse.

           - Call `run_validator(run_dir, phase, gh)`. Get ValidatorResult.
           - IMPORTANT: Budget check FIRST (Pitfall 2). Check budget again after getting result. If breached, transition to EXIT with EXHAUSTED.
